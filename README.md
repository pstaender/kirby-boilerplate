# Your project

## Opionionated Boilerplate

Uses:

    * yarn (Node v16+)
    * composer (PHP v7.3+)

Features:

    * theme-folder
    * scss, es5+
    * parcel v2 bundler (includes a no-hash-namer)
    * svg sprite generator
    * kirby cms (replace with your choice of framework / cms)

Modules:

    * include-media
    * normalizer
    * modernizr

## Install dependencies

    $ composer install
    $ yarn install

## Development mode

    $ npm run develop

## Build for production

    $ npm run build
